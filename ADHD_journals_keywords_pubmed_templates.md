# ADHD Journals + Keywords + PubMed Search Templates
_Last updated: 2026-04-13_

## Purpose
This file is a practical English reference for **ADHD-related medical, psychology, and neuroscience literature searching**.  
It combines:

- journal names
- topic positioning / scope
- recent **Q1–Q2 status** (category- and year-dependent; always re-check before submission)
- common ADHD-related keywords
- ready-to-use **PubMed search templates**

## Notes
- Quartiles can vary by **database, subject category, and year**.
- The quartile labels below are meant as **practical recent-reference labels** for literature scouting and journal targeting.
- PubMed templates use a broad ADHD core:
  - `"Attention Deficit Disorder with Hyperactivity"[Mesh]`
  - `ADHD[tiab]`
  - `"attention-deficit/hyperactivity disorder"[tiab]`
- You can swap topic blocks depending on your interest: diagnosis, executive function, medication, neuroimaging, comorbidity, adults, females, sleep, emotion regulation, etc.
- In PubMed, journal searching can be done with the journal title tag **[ta]**.

## Simple Editing Rule
To narrow results fast, append one of these blocks to any template:

- `AND (systematic review[pt] OR meta-analysis[pt])`
- `AND (randomized[tiab] OR trial[tiab])`
- `AND (child*[tiab] OR adolescen*[tiab])`
- `AND adult*[tiab]`
- `AND english[la]`
